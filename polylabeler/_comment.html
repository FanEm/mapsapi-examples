Пример использования модуля для подписи полигонов.

Модуль позволяет добавлять подписи внутри полигонов на карте, автоматически выбирая подходящее положение. Для использования модуля необходимо скачать исходный код модуля для расчета площади, модуля для подписи полигонов и положить в директорию вашего сайта.

Для многоугольника создаются две подписи: маленькая точка и полноразмерная подпись. Если полноразмерная подпись не может вписаться в многоугольник, модуль пытается разместить маленькую точку. Если маленькая точка не помещается, то ничего не отображается.
